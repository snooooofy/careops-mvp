CareOps MVP - Next.js scaffold

This repository is a minimal Next.js scaffold for the CareOps Hackathon MVP. It includes:

- Basic pages: index, onboarding, dashboard, inbox, bookings
- Simple API health endpoint

To run locally:

1. cd careops-mvp
2. npm install
3. npm run dev
